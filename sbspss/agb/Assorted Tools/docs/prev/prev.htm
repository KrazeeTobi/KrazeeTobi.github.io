prev.sh [Tool Command]

Previews animation according to the script.

Syntax

prev.sh script-file

Description

Lets you preview the complicated animations described with a script. Executing this command makes an objprev.bin AGB execution file that you can send to the emulator.

The command arguments are described below:

script-file
The text file describing the script. A preview file will be created after analysis of this file.

Notes

You cannot omit the ".sh" part of the command. This application operates on "bash." Using this command you can preview flip books and a variety of rotated and scaled animations. Jump here to learn about the flow of commands.
