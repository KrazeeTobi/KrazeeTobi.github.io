Previewer - Samples

1. Description

A number of samples have been prepared in order to help you gain a better understanding of the Previewer. These include sample1.txt, sample2.txt, sample3.txt, sample4.txt and anim_sample.txt in the Previewer folder and the "sample" folder images.

2. How to Use the Samples

sample1.txt -- A simple object sample
sample2.txt -- This sample uses the include file
sample3.txt -- A BG sample
sample4.txt -- A composite sample of objects and BG

For sample1.txt and sample3.txt, enter separately as
>prev.sh sample1.txt
>prev.sh sample3.txt
to make an agbprev.bin AGB execution file in the same folders, which you then send to the emulator.
For sample2.txt and sample4.txt, first enter
>prevprep sample2.txt -o sample_tmp.txt
>prevprep sample4.txt -o sample_tmp.txt
to pass through the preprocessor, and then execute
>prev.sh sample_tmp.txt
This command makes an agbprev.bin file as mentioned above, which you then send to the emulator.

The latter two files reference anim_sample.txt for the animation data. The preprocessor incorporates this into the sample. The same file is referenced, so in sample2 and sample4 different characters are animated the same way.
