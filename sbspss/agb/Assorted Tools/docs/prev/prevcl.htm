prevcl [Tool Command]

Converts a script into a C language source file.

Syntax

prevcl [-o/-d] script-file

Description

Converts into a C language source file the script file that describes the contents of an animation. This command is called from prev.sh.

The command arguments are described below:

script-file
The text file describing the script. This file is analyzed, corrected to match the AGB format and then output as a C language source file.
-o output-file
Specifies the name of the output file. However, since the data is output as header file and source file, the filename extension is converted to .h .c automatically.
-d
Outputs the dependencies. When this argument is specified, the extension of the output file is not converted to .h .c.

Notes

The C source file created with this command is copied to the folder containing AGB previewer source files, after which, "make" is done to create the Previewer itself. These tasks are executed inside prev.sh.
