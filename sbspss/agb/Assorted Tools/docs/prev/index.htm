AGB Previewer Reference


About the Previewer

Commands

prev.sh The command for previewing a script.
imgview.sh The command for previewing an image.
fbview.sh The command for previewing a flip book.

Sub-commands

prevprep The C language-like preprocessor for the script.
prevcl Converts the script into C language source.
bmp2prev Creates a script in order to preview an image.
bmp2fb Creates a script in order to preview a flip book.
