AGB Tools Command Reference


act2agb A tool for converting the .act palette files output with Photoshop into C source.
bmpgrid A tool for converting a Windows Bitmap into a number of bitmaps. Can be cut to any size, providing it is a multiple of 8. Supports 4bit, 8bit and 24bit formats.
sgi2bmp A tool for converting Silicon Graphics formats into Windows Bitmap. Supports any format that does not use palettes.
bmp2bin Converts Windows Bitmap dot data to a binary file. Has both a bitmap mode and a character mode, converting both formats into binary. Supports 4bit, 8bit and 24bit formats.
agbparts Takes map data stored as a Windows Bitmap and divides it into parts. Supports 4bit and 8bit formats.
bmp2map Takes map data stored as a Windows Bitmap, divides it into parts, and also outputs the generated map data as C source.
bin2h This tool produces a header file listing the array names that are created when binary files are linked.
