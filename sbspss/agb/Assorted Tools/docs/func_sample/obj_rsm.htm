Functional samples: OBJECT FUNCTIONS

This is a sample of objects.

Source: (AGBDIR)\src\samples\functional_sample\obj_rsm\

Description

This is a sample of object-related AGB functions. It covers everything from basic motions to scaling-rotation and mosaics. You can observe these effects by toggling on/off the character double-size field used when scaling and rotating.

How to operate

SELECT button ... Return to Title Menu
START button ... Initialize scaling and all other parameters
Control Pad ... Move
LR buttons ... Alter the mosaic
Control Pad while pressing the B button ... Scale
LR buttons while pressing the B button ... Rotate
A button ... Switch the character double-size field
