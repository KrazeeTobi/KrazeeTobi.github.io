Functional samples: BG FUNCTIONS

This is a sample of BG functions.

Source: (AGBDIR)\src\samples\functional_sample\bg_rsm\

Description

This is a sample of BG-related AGB functions. It covers motion, scaling, rotation and mosaics in scale/rotate mode.
The cloud in front is the subject of manipulation. The cloud has been made semi-transparent. To learn about this, see the ALPHA BLEND sample. The background surface and the animated ball cannot be moved.

How to operate

SELECT button ... Return to Title Menu
START button ... Initialize scaling and all other parameters
Control Pad ... Move
LR buttons ... Alter the mosaic
Control Pad while pressing the B button ... Scale
LR buttons while pressing the B button ... Rotate
