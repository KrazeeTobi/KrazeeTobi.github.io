Running mks4agb.exe

Run mks4agb.exe using one of the following methods. 

(1) To run from a  "MS-DOS Prompt (DOS Window)," if the current directory is the Work Directory,

Type the following command.
        >mks4agb [enter]

If not, type the following command.
        >mks4agb Working Directory [enter]

(2) To run right from an icon, place all of the MKS4AGB system files in the Work Directory and run mks4agb.exe by double clicking or some other means. 

(3) To run from a Short Cut Icon, create a "short cut to mks4agb.exe." Open the short cut properties and specify the  Work Directory in the "Working Folder." Run the file by double-clicking or some other means. 

In this case, the directories containing aif2agb.exe and mid2agb.exe must also be in the executable path.
