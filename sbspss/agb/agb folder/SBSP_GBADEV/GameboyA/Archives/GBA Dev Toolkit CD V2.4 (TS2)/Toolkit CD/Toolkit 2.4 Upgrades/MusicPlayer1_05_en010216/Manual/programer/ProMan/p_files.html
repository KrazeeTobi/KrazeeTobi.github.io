Description of Files

Game programmers will receive the following types of files from sound engineers.
     {wavename}.o       Waveform Data Object Files (More than one)
     {songname}.o        Song Data Object Files (More than one)
     SoundDat.o            Other Data Object Files
     m4aLib.o                Sound Function Library Code
     m4aLib.h                Sound Function Include File
     Soundfiles              List of All Sound Object Files
