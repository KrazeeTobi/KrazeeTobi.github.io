Direct Sound Simultaneously-Produced Sounds and Playback Frequencies

Direct sound allows you to increase the number of simultaneously-produced sounds to as many as the memory and speed of the hardware will allow. With this driver you can set from 1-12 sounds.

Additionally, even if the direct sound playback frequencies with this driver continue on forever, you are only allowed to select the following values so a conflict does not develop with the V-Blank cycles. 

     5734 Hz,  7884 Hz, 10512 Hz, 13379 Hz, 15768 Hz, 18157 Hz,
    21024 Hz, 26758 Hz, 31536 Hz, 36314 Hz, 40137 Hz, 42048 Hz
Note: Increasing the number of simultaneously-produced sounds and/or the playback frequencies increases the processing time.
