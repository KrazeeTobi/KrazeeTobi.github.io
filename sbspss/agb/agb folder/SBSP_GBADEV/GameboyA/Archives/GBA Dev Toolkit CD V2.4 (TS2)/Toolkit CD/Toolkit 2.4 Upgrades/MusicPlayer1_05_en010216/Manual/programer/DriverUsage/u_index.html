Using AGB Direct Sound Driver

Overview
  This manual describes how to use the direct sound driver associated with the Nintendo's
  Game Boy Advance monitor ROM.


Cautions
  -System calls are used with the direct sound driver. Therefore, this explanation is
  intended for users of the Development AGB (TS4).
  -You cannot use this along with MP2000.
  -This is slower and unstable compared to the newest version of MP2000.
*The referencing of variables which are not described is not supported.
