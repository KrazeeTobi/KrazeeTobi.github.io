Number of Simultaneously-Produced Direct Sounds

Direct sounds make it possible to produce as many sounds simultaneously as the hardware (memory and processing speed) will allow, but with this system, you can set the values of the 1st through 12th sounds. The values of these settings apply to all games, so you will not be able to set up the simultaneously produced sounds on individual Music Players. Conceptually, imagine a configuration where all of the Music Players are sharing a single synthesizer.

Also, if you try to exceed the number of simultaneously-produced sounds, the songs with the higher priority settings will be played first. Within the same song, the lowest track numbers will be played first.
(See also Priorities.)

Note: The processing time will increase if you increase the number of simultaneously-produced sounds.

During a state of excessive processing, there will be noise in the sounds produced, so we recommend considering how many sounds to produce for each situation.
