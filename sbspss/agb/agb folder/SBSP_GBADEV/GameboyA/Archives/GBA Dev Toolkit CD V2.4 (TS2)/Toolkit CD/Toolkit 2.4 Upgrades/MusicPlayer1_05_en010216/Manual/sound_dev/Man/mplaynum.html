Number of Music Player Tracks

When several Music Players have been opened, each will set aside the number of tracks that it requires. As the number of tracks increases, the processing time will not be affected too much, but the amount of Working RAM used will increase. 

Also, avoid confusing the term "Number of Tracks," with the term "Number of Simultaneously-Produced Sounds" described below, as they have nothing to do with each other. 
To draw a comparison, the number of tracks is like the number of staves (or parts) on a sheet of music. The number of simultaneously-produced sounds is comparable to the number of sounds that a synthesizer can produce (polyphonous).
