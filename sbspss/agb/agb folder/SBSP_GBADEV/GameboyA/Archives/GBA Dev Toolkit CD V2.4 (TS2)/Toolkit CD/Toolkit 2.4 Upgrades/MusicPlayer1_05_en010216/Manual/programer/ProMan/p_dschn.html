Number of Simultaneously-Produced Direct Sounds

It is possible to increase the number of simultaneously-produced direct sounds to the limits of the AGB speed and memory. However, in this system, the values for sounds 1 through 12 can be assigned.

Note: Increasing the number of simultaneously-produced sounds will increase the processing time.

Excessive processing will generate noise in the sounds produced. We recommend considering what would be an appropriate number of sounds to produce for each project developed.
