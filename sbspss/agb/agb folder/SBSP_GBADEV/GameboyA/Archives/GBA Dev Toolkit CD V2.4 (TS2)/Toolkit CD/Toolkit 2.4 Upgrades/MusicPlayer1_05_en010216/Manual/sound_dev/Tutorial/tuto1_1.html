Viewing the Demonstration

Demonstration Screen


The tutorial comes equipped with a musical demonstration created using the mks4agb system.

Download and unzip the following file.  Try running it on the emulator.
The name of the executable file is mp2kdemo.bin.

* Download the MUSIC PLAYER AGB 2000 Demonstration (mp2kdemo.zip 164KB)

After the above screen has been displayed, the demonstration, in which Wario flies around in an airplane, will begin. This tutorial will explain how to make the background music and sound effects used in this demonstration, in addition to giving an overview of the mks4agb system sound drivers.
