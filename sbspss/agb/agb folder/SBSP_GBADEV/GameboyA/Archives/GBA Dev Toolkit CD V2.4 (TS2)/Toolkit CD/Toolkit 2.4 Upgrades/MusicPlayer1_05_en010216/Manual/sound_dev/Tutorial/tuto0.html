Introduction
This Tutorial explains sound development using the mks4agb system. 
We have prepared a demonstration in which Wario flies around in an airplane with background music and sound effects playing. 
The MIDI and AIFF files used to create this demo are also available, and we will describe how to use the mks4agb system while actually using it. 
At the end, you will be able to verify the same sounds as the demonstration using the simple monitor.
