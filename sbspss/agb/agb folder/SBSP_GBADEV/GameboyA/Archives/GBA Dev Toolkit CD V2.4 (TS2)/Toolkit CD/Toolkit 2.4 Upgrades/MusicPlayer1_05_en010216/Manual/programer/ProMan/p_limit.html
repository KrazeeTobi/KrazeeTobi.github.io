Maximum Processing Time for m4aSoundMain( )

The processing time for m4aSoundMain( ) for each 1/60 second frame can be limited by the approximate number of scan lines. This setting value is the amount of time allowed for processing in lines - not the location where processing is stopped when scan lines reach it. It is impossible to accurately limit time so in reality, the setting value is exceeded by a certain number of lines.

In addition, if you want to set a limit for the processing time for m4aSoundMain( ), you need to have the sound engineer change the max_line value in the definition file mks4agb.ini.
