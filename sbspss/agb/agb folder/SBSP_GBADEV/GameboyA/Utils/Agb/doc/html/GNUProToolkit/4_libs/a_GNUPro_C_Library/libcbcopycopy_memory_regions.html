bcopy
[copy memory regions] 

SYNOPSIS 
#include <string.h> 
void bcopy(const char *in, char *out, size_t n);

DESCRIPTION
The function, bcopy, copies n bytes from the memory region pointed to by in to the memory region pointed to by out. This function is implemented in term of memmove.

RETURNS
bcopy does not return a result.

COMPLIANCE
bcopy requires no supporting OS subroutines.
