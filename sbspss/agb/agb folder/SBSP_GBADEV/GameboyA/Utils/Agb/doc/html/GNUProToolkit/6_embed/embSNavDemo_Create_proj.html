Creating the Source-Navigator demonstration project

With Source-Navigator active, create a new project.
1.
Choose File -> New Project... from the Symbol Browser and define a new project, calling it monop.proj.
2.
Under Add Directory, click the “...” button to select the demos\monop directory. See Figure 1 for the result of defining the new project’s name in the Auto-Create Project window. Click OK to build the project.
