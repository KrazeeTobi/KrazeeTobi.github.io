strstr
[find string segment] 

SYNOPSIS 
#include <string.h> 
char *strstr(const char *s1, const char *s2);

DESCRIPTION s
trstr locates the first occurrence in the string pointed to by s1 of the sequence of characters in the string pointed to by s2 (excluding the terminating null character).

RETURNS
Returns a pointer to the located string segment, or a null pointer if the string s2 is not found. If s2 points to a string with zero length, the s1 is returned.

COMPLIANCE
strstr is ANSI C.

strstr requires no supporting OS subroutines.
