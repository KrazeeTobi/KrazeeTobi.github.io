 Known causes of trouble with GNU CC  

The following documentation describes known problems that affect users of GNU CC.
  • Actual bugs we haven’t fixed yet
  • Iinstallation problems
  • Cross-compiler problems
  • Interoperation
  • Problems compiling certain programs
  • Incomaptibilites of GNU CC
  • Fixed header files
  • Standard libraries
  • Disappointments and misunderstandings
  • Common misunderstanding with GNU C++
    • Declare and define static members
    • Temporaries may vanish before you expect
  • Caveats of using protoize
  • Certain changes we don't want to make
  • Warning messages and error messages
Most of the problems are not GNU CC bugs per se—if they were, we would fix them. But the result for a user may be like the result of a bug.

Some of these problems are due to bugs in other software, some are missing features that are too much work to add, and some are places where people’s opinions differ as to what is best.
