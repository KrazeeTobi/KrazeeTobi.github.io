memset
[set an area of memory] 

SYNOPSIS 
#include <string.h> 
void *memset(const void *dst, int c, size_t length);

DESCRIPTION
The function, memset, converts the argument c into an unsigned char and fills the first length characters of the array pointed to by dst to the value.

RETURNS
memset returns the value of m.

COMPLIANCE
memset is ANSI C.

memset requires no supporting OS subroutines.
