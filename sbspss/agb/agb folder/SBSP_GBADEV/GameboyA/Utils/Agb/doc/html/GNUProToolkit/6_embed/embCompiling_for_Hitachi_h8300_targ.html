Compiling for Hitachi H8/300 targets  

The Hitachi H8/300 target family toolchain controls variances in code generation directly from the command line. When you run GCC, you can use command-line options to choose whether to take advantage of the extra Hitachi H8/300 machine instructions, and whether to generate code for hardware or software floating point.  See also Compiler options for the Hitachi H8/300, H8S and H8/300H.
