Getting in and out of GDB  

The following documentation discusses the essentials of GDB: the invoking the debugger, choosing files, choosing modes, stopping a process and shell commands.
Primarily, to start GDB and quit GDB, use the following instructions.
  • Type gdb to start the debugger in a graphical interface mode or use the command, gdb -nw, to start the debugger in a non-window interface mode.
  • Type quit or use the keystroke sequence, Ctrl-d, to exit.
