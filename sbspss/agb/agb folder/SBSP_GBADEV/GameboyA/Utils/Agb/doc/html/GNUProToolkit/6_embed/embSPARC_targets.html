SPARC targets 
The GNUPro Toolkit supports both the SPARC and the Fujitsu SPARClite families. For the compiler in particular, special configuration options allow you to use special software floating-point code for the SPARC MB86930 chip, as well as defaulting command-line options to use special Fujitsu SPARClite features.

For the Fujitsu SPARClite, the GNUPro Toolkit currently supports boards ex930, ex931, ex932, ex933, ex934, and ex936.
