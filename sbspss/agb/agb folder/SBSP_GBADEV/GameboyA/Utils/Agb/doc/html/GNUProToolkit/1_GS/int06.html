Cygnus InsightTM
Cygnus InsightTM, the visual debugger for GNUPro Toolkit, launches, displaying the Source Window (see its accompanying descriptions, Using the Source Window). Using the Source Window’s View menu or the tool bar buttons, display Insight’s other windows.
 
For a demonstration of interoperability of Insight and Source-Navigator, the source code comprehension tool, see Starting Insight from Source-Navigator and Source-Navigator demonstration.
