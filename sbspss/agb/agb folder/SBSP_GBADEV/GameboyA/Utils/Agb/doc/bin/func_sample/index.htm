How to Operate the Title Menu

SELECT button ... Aligns the cursor one menu below. If cursor is at the bottom, it returns to top.
START button ... Starts the sample
Control Pad UP/DOWN ... Selects a sample (the same when explanations are displayed)
LR buttons ... Switches menu pages
B button ... Returns to Menu Title when an explanation is being displayed.
A button ... Displays the explanation in order and then starts the sample.
