Selecting the target system
You can specify three aspects of the target system to the GNU binary file utilities, each in several ways:
  • the target; see discussions for Target selection.
  • the architecture; see discussions for Architecture selection.
  • the linker emulation, applying to the linker only; see discussions for Linker emulation selection .
In the discussions for Target selection, the lists of ways to specify values are in order of decreasing precedence for each object file format.

The commands to list valid values only list the values for which the programs you are running were configured. If they were configured with
‘--enable-targets=all’, the commands list most of the available values, but a few are left out; not all targets can be configured in at once because some of them can only be configured native (on hosts with the same type as the target system).
