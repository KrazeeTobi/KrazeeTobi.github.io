Readline movement commands
The previous commands are the most basic possible keystrokes that you need in order to do editing of the input line. For your convenience, many other commands have been added in addition to C-b, C-f, C-d, and Delete. The following documentation discusses some commands for moving more rapidly about the line.
    C-a
      Move to the start of the line.
    C-e
      Move to the end of the line.
    M-f
      Move forward a word.
    M-b
      Move backward a word.
    C-l
      Clear the screen, reprinting the current line at the top.
Notice how C-f moves forward a character, while M-f moves forward a word. It is a loose convention that control keystrokes operate on characters while meta keystrokes operate on words.
