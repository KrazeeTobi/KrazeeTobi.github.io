iprintf
[write formatted output (integer only)]

SYNOPSIS
#include <stdio.h>
int iprintf(const char *format, ...);

DESCRIPTION
iprintf is a restricted version of printf: it has the same arguments and behavior, save that it cannot perform any floating-point formatting. The type specifiers, f, g, G, e, and F, are not recognized.

RETURNS
iprintf returns the number of bytes in the output string, save that the concluding NULL is not counted. iprintf returns when the end of the format string is encountered. If an error occurs, iprintf returns EOF.

COMPLIANCE
iprintf is not required by ANSI C.
Supporting OS subroutines required: close, fstat, isatty, lseek, read, sbrk, write.
