Static pattern rules
Static pattern rules are rules which specify multiple targets and con-struct the dependency names for each target based on the target name. They are more general than ordinary rules with multiple targets be-cause the targets do not have to have identical dependencies. Their dependencies must be analogous but not necessarily identical.
