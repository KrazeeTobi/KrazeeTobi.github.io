difftime
[subtract two times] 

SYNOPSIS 
#include <time.h> 
double difftime(time_t tim1, time_t tim2);

DESCRIPTION
difftime subtracts the two times in the arguments: tim1- tim2.

RETURNS
The difference (in seconds) between tim2 and tim1,as a double.

COMPLIANCE
ANSI C requires difftime, and define its result to be in seconds in all implementations.

difftime requires no supporting OS subroutines.
