Functional samples: BITMAP FUNCTIONS

This is a sample of the BG bitmap modes (3 to 5).

Source: (AGBDIR)\src\samples\functional_sample\bmpmode\

Description

This is a sample of functions relating to the BG bitmap modes.

This sample uses BG modes 3 to 5 to display images as follows:

BG mode 3: Bitmap of 240x160 dots and 65536 colors
BG mode 4: Bitmap of 240x160 dots and 256 colors
BG mode 5: Bitmap of 160x128 dots and 65536 colors

How to operate

SELECT button ... Return to Title Menu
A button ... Switch the BG mode
R button ... Shrink image
L button ... Enlarge image
Control Pad ... Move image
START button ... Return scaling parameter to initial value
