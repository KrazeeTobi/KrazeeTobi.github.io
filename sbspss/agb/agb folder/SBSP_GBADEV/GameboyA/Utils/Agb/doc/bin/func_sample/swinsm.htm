Functional samples: WINDOW FUNCTIONS

A sample of simple windows.

Source: (AGBDIR)\src\samples\functional_sample\swinsm\

Description

This is a sample of the functions of Window 0, Window 1 and OBJ windows.
AGB can have two rectangular windows and an optional number of OBJ windows. It is possible to set display and color special effects ON/OFF separately, for the inside of Window 0, inside of Window 1, inside the OBJ windows, and outside the windows.
In this sample program, different settings are made for the objects displayed inside three windows, so that each window shows something different.
Window type BG0 BG1 BG2 BG3 OBJ Special effects
Window 0 O O O O
Window 1 O O O
OBJ Window O O O O O
Outside Windows O O O O O O

How to operate

SELECT button ... Return to Title Menu
Control Pad ... Move Yoshi
Control Pad while pressing A button ... Move OBJ window
START button ... Change Yoshi priority. 0 -> 1 -> 2 -> 3 -> 0 -> ...
