 Non-constant initializers  

As in standard C++, the elements of an aggregate initializer for an automatic variable are not required to be constant expressions in GNU C. The following is an example of an initializer with run-time varying elements. 
    foo (float f, float g) 
{ 
   float beat_freqs[2] = { f-g, f+g }; 
   ... 
}
