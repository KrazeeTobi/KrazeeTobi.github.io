agbparts [Tool Command]

Divides out parts from a Bitmap assembled from 8x8-dot parts.

Syntax

agbparts [-f] -o output-file input-file

Description

If you input a Bitmap image assembled from 8x8-dot parts, the image will be disassembled into parts and optimized (identical parts counted as one), and then Bitmaps of the parts will be output.

The command arguments are described below:

-o output-file
Specifies the bitmap filename for output
-f
Parts flipped vertically or horizontally are treated as identical. This argument is set automatically if a 4bit Bitmap is input.

Notes

Supports the 4bit, 4bit RLE-compressed, 8bit and 8bit RLE-compressed image formats. Parts are constructed in the order in which they appear. Parts are treated as separate entities even if they differ by only one dot. After the command is executed, a message is displayed in the console saying something like: "Number of characters used: 200." Please reference this information when creating maps.
