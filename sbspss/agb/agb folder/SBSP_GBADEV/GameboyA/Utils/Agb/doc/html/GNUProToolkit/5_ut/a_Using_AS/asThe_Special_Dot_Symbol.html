The special “dot”  symbol, “.”
The special “.” symbol refers to the current address into which as is assembling.

Thus, the expression, melvin: .long, defines melvin to contain its own address.

Assigning a value to “.” is treated the same as a .org directive. Thus, the expression of ‘.=.+4’ is the same as saying ‘.space 4’.
