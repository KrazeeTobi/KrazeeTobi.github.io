memcmp
[compare two memory areas]

SYNOPSIS
#include <string.h>
int memcmp(const void *s1, const void *s2, size_t n);

DESCRIPTION
The function, memcmp, compares not more than n characters of the object pointed to by s1 with the object pointed to by s2.

RETURNS
The function returns an integer greater than, equal to or less than zero, according to whether the object pointed to by s1 is greater than, equal to or less than the object pointed to by s2.

COMPLIANCE
memcmp is ANSI C.

memcmp requires no supporting OS subroutines.
