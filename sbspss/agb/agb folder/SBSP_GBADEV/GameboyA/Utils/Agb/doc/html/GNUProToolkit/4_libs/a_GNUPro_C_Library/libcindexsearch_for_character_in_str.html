index
[search for character in string] 

SYNOPSIS 
#include <string.h> 
char *index(const char *string, int c);

DESCRIPTION
The function, index, finds the first occurrence of c (converted to a char) in the string pointed to by string (including the terminating null character).

This function is identical to strchr.

RETURNS
Returns a pointer to the located character, or a null pointer if c does not occur in string.

COMPLIANCE
index requires no supporting OS subroutines.
