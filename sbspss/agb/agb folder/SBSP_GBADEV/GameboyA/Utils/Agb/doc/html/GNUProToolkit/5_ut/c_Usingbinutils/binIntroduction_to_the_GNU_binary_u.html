Introduction to the GNU binary utilities
This documentation contains basic descriptions for the following GNU binary utilities:
 
    ar
      To create, modify, and extract from archives
    nm
      To list symbols from object files
    objcopy
      To copy and translate object files
    objdump
      To display information from object files
    ranlib
      To generate index to archive contents
    size
      To list file section sizes and total size
    strings
      To list printable strings from files
    strip
      To discard symbols
    c++filt
      To demangle encoded C++ symbols
    addr2line
      To convert addresses into file names and line numbers
    nlmconv
      To convert object code into a Netware Loadable Module (NLM)
    windres
      To manipulate Windows resources
