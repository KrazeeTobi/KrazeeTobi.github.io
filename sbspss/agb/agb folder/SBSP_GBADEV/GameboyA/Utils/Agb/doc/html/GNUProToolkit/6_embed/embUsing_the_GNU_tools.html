Using the GNU tools on embedded systems 
The following GNUPro Toolkit tools can be run an embedded target.
  • gcc, the GNU compiler
  • cpp, the GNU preprocessor
  • as, the GNU assembler
  • ld, the GNU linker
  • binutils, the GNU binary utilites
  • gdb, the GNU debugging tool
  • libgloss, newlib AND libstd++, the GNU libraries
  • crt0, the main startup file
  • The linker script
  • I/O support code
  • Memory support
  • Miscellaneous support routines
