Functional samples: COLOR SPECIAL EFFECT

This is a sample of color special effects.

Source: (AGBDIR)\src\samples\functional_sample\coleffsm\

Description

This is a sample of color special effects.
In this sample you can change the type of color special effects and turn OBJ semi-transparency ON/OFF.
BG0 is on the ground to the bottom-right, BG1 is in the air to the top-left, and the three Yoshis starting from the left are OBJ0 and OBJ1 and OBJ2.
The part falling off the lower-right side is a window, inside which the color special effects are OFF.

When OBJs are made semi-transparent and the color special effect is set to Luminosity UP or Luminosity DOWN, you will see no change in luminosity for parts of the OBJ that are semi-transparent. But you will notice a change in luminosity for those parts that are not semi-transparent (where the pixels directly underneath are not the second object).

Color special effects are OFF inside the Window, so they will have no effect.
However, a semi-transparent OBJ on top of the second object will be semi-transparent.
If there is no second object, semi-transparency will not arise.
Even if a moving OBJ overlaps a fellow OBJ, that fellow OBJ will not become semi-transparent.

How to operate

SELECT button ... Return to Title Menu
START button ... Change the special effect setting (Normal -> Alpha blend -> Luminosity UP -> Luminosity DOWN)
START button while pressing B button ... Switch OBJ semi-transparency ON/OFF
Control Pad ... Move Yoshi 0
Control Pad while pressing B button ... Move Yoshi 1
Control Pad while pressing A button ... Move Yoshi 2
L button ... Make EVA -1 and EVB +1
R button ... Make EVA +1 and EVB -1
L button while pressing B button ... Make EVY -1
R button while pressing B button ... Make EVY +1
(EVA = coefficient for 1st (upper) object of alpha blend; EVB = coefficient for 2nd (lower) object of alpha blend; EVY = coefficient for change in luminosity.)
