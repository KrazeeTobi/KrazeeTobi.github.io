strlwr
[force string to lower case]

SYNOPSIS
#include<string.h>
char *strlwr(char *a);

DESCRIPTION
strlwr converts each characters in the string, at a, to lower case.

RETURNS
strlwr returns its argument, a.

COMPLIANCE
strlwr is not widely portable.

strlwr requires no supporting OS subroutines.
