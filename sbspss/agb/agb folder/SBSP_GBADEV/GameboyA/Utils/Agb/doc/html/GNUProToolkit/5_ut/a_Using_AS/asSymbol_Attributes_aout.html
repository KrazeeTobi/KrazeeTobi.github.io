Symbol attributes: a.out
The following documentation discusses the symbol attributes for the a.out format

Descriptor

This is an arbitrary 16-bit value. You may establish a symbol’s descriptor value by using a .desc statement (see .desc symbol, abs-expression ). A descriptor value means nothing to as.

Other

This is an arbitrary 8-bit value. It means nothing to as.
