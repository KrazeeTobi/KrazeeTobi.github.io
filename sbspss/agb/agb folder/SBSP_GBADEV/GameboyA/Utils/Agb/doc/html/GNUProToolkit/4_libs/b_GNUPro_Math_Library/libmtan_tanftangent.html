tan, tanf
[tangent]

SYNOPSIS
#include <math.h> 
double tan(double x); 
float tanf(float x);

DESCRIPTION
tan computes the tangent of the argument, x. Angles are specified in radians.

tanf is identical, save that it takes and returns float values.

RETURNS
The tangent of x is returned.

COMPLIANCE
tan is ANSI. tanf is an extension.
