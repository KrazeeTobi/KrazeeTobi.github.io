 Compiling for SPARC targets 
The SPARC target family toolchain controls variances in code generation directly from the command line.

When you run GCC, you can use command-line options to choose whether to take advantage of the extra SPARC machine instructions, and whether to generate code for hardware or software floating point. See also Compiler options for SPARC.
