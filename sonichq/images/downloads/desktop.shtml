Sonic Desktop
 

In here, you'll find Sonic desktop themes, wallpaper, and other things that make your computer desktop look way Sonic! Most things here are from SEGA or converted from something made by SEGA. Soon, I'm going to add links to freeware programs that will come in handy when using themes.


Themes

The Sonic Theme Collection!
A set of themes complied by me, Vector. After I downloaded Andy Watt's theme, I started customizing it and eventually added themes for other characters, too. I've included my customized themes, which include Sonic, Tails, Knuckles, Vector, Mighty, SegaSonic the Hedgehog, and 2 Sonic & Knuckles thems, in this zip. This desktop theme collection is 1.53MB in size.


The Sonic the Hedgehog desktop theme by Andy Watts (Email / Website). It contains things such as wallpaper, icons, sounds, and more! This desktop theme is 552KB in size.
