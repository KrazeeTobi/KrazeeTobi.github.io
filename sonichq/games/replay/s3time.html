Sonic 3 Time Attack Challenge


Introduction and Updates:

If you have Sonic 3, please send me your best times for the Time Attack mode. To see them, go to Competition and select Time Attack, then finish one of the zones to see the 3 best times, along with the characters who got them.

Since every category has openings, any valid time will make it onto the chart.


Include:

1. Any times you have that are good enough to go on the chart. Since there are currently openings in every category, that means any time that you have!
2. The character you used to get each time.
3. The name you want to use.
4. A color for your name. It can be a word (blue) or a hexadecimal code (0000FF).
5. If you want, an e-mail address or website that your name will link to.


Rules:

1. Don't send any times that you got while using a code.
2. Of course, you shouldn't send any times that you made up.
3. Use only your Time Attack results, not results from Match Race or Grand Prix.
4. You can't use the "off-screen" trick in the Balloon Park Zone. See the Games List if you don't know what I mean.
