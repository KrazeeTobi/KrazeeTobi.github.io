Posting Tools

Welcome to the Posting Tools home page! This 3rd-party script is designed to add additional posticons and other features to EZBoards. It's written in javascript and requires no use of site permissions, just a file on your server and some code in your board header. I may make a remotely-hosted version, which would require only a code in your board header and include automatic updates, if enough people ask for this feature. Be sure to check back for updates and new features. To comment on this script, post here. (You can also see the script (or at least one similar to it) running there.) Note - this script is IE5+ only.
