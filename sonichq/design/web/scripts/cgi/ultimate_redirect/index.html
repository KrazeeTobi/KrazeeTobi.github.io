About This Script
Ultimate Redirect lets you redirect to multiple sites using a directory-style URL. Whatever is added to the URL of the script, as long as it starts with a ? or /, will be included in the code for the site. You can match codes with sites within the script itself and include a default URL for incorrect codes.

Example
redirect.cgi/hq
redirect.cgi/hq/designhq

Download
Ultimate Redirect - .3 KB

Save the file as redirect.cgi. Standard CGI installation instructions apply.

Tips & Tricks
This script is useful for files that you link to frequently. If you link to the redirector instead of the file's real URL, you can move the file or rename it and you'll only have to upate the URL in the redirect script instead of links on all of your pages.
