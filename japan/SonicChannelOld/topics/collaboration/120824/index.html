初音ミク Project DIVA Arcade 追加モジュール！

2012.08.24

現在、アーケードで稼働中のセガのリズムアクション『初音ミク Project DIVA Arcade』にて、ソニックチームの上川祐司が手がけたモジュール「ソニックスタイル」が2013年8月31日に追加されます。

『初音ミク Project DIVA Arcade』追加モジュールとは、ゲームに登場するキャラクターの服装・見た目などを変更できる要素です。モジュールの追加には、ゲームでVOCALOID POINTを貯める等して増やす事が出来ますよ。

ソニックに扮した初音ミクを、筐体の設置してあるお店でぜひ確認してみて下さいね！
