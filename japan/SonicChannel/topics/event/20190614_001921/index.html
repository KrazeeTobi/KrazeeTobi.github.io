SONIC DJ SET - BIRTHDAY MIX（2019）Youtubeプレミア公開！【06/19 UPDATE】

2019.06.14

ソニックシリーズサウンドディレクター
大谷智哉

メガドライブ用ゲームソフト『ソニック・ザ・ヘッジホッグ』が発売してから28年を迎える2019年6月23日(日)、ソニック・ザ・ヘッジホッグ生誕28周年を記念して、ソニック日本公式YouTubeチャンネルにてソニックバースデー特別映像をプレミア公開（※）いたします。

この動画公開では、「ソニック」シリーズサウンドディレクターの大谷智哉による、ソニック生誕28周年をお祝いするソニック楽曲オンリーのDJプレイの様子をご覧いただけます。歴代のゲームシリーズからセレクトされた楽曲を、約1時間にわたるノンストップミックスでお届けします。公開は6月23日(日)16:00から開始の予定です。

※Youtubeプレミア公開は、公開時間の決まっている動画公開となり、全世界で同じ時間帯に映像体験を共有できます。

東京ジョイポリスにてライブビューイング

また、本映像は6月23日(日)16:00の同時刻、東京ジョイポリス（東京／お台場）の3rd Floorマルチステージにて、ライブビューイングでの放映が決定しました。東京ジョイポリスでは同日、6月23日(日)の一日限定で、ソニック28回目のバースデーを祝して特別なフードメニューやキャラクターグリーティングイベントなどを実施します。この機会にぜひ、東京ジョイポリスへお越しください。

詳しくは、東京ジョイポリス公式ホームページをご確認ください。

懐かしのあの曲から最新のあの曲まで、ソニック楽曲オンリーのDJミックスを聴きながら、ソニック28回目のバースデーを全世界のファンの皆さまとともにお祝いしましょう。
