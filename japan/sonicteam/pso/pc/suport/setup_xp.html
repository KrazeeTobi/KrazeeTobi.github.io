【各OSごとの簡体字中国語・繁体字中国語・韓国語の表示に関して】
■Windows XP
Windows XP は標準で、簡体字中国語・繁体字中国語・韓国語がインストールされております。ユーザーが任意でこれらのフォントを削除してしまった場合は、以下の手順でインストールしてください。
１． Windows XPのDISCをCDドライブに入れてください。
２．
Windowsの[スタート] メニューから、[コントロール パネル]を選択し、開いてください。
３． コントロールパネル内の「日付、時刻、地域と言語のオプション」を選択し、開いてください。
４． 「日付、時刻、地域と言語のオプション」内の「ほかの言語を追加する」又は「地域と言語のオプション」を選択してください。
５． [詳細設定] タブのを選択し、[コードページ変換テーブル] でインストールをしたい言語(韓国語・簡体字中国語・繁体字中国語)のチェック ボックスをオンにして「ＯＫ」ボタン

を押せばフォントがインストールされます。
６． フォントのインストールが完了しました。設定を反映させるためにXPを再起動してください。
