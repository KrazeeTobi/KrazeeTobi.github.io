TOKYO GAME SHOW 2012 レポート

2012年09月21日

みなさんこんにちは！
ソニックチャンネル スタッフ★です。



じゃじゃーん！今年もやってきましたTOKYO GAME SHOW！
今回もTOKYO GAME SHOW 2012 ビジネスデーでのセガブース、現地レポートをお届けしますよ☆
一般日22日、23日に行かれる際に参考にして下さいね。


★セガスタジオ


まずは、毎日様々なイベントが盛りだくさんのセガスタジオブース

気になるスタジオプログラムは、セガのTOKYO GAME SHOW2012のステージイベントページをチェック

★龍が如く
セガスタジオブース横には、龍が如くスタジオの看板タイトル「龍が如く」シリーズの記念すべき一作目と二作目のHDバージョン 『龍が如く 1&2 HD EDITION』の壁画！圧巻の迫力
『龍が如く 1&2 HD EDITION』の壁画を含めた、このブースは 『龍が如く5 夢、叶えし者』がプレイできるブースとなっています。
入口では、バーカウンターでお姉さんたちがお出迎えしてくれますよ


★SEGA_Apps
『龍が如く 夢、叶えし者』ブースの裏側は、SEGAがお送りするスマホアプリが勢揃いのSEGA_Appsブースです。

様々なアプリが出展中ですが、その中には以前ソニックタイトルのディレクションをした中村俊がプロデュースし、ニンテンドー3DSで発売された『リズム怪盗R 皇帝ナポレオンの遺産』が『リズム怪盗R for iOS(仮)』となって展示してありますので、チェックしてみて下さ〜い。


★セガパートナーズ
SEGA_Appsブースを超えた先は、セガパートナーズブース。コードマスターズブースでは『F1 RACE STARS』『F1 2012』が出展中

続いて同じくセガパートナーズEAブースでは、『FIFA 13』『MOST WANTED』『MEDAL OF HONER WARFIGHTER』がそれぞれのブースで出展中




★ファンタシースターオンライン2
その奥を進むと迫力の巨大ドラゴンがお目見え
『ファンタシースターオンライン2』ブースではPC/PS Vita/スマートフォンが出展中です
『ファンタシースターオンライン2』ブースには、酒井プロデューサーがいたので『ファンタシースターオンライン2』の簡単な紹介とソニックファンに向けて一言頂いてきました。


■酒井
今回のTOKYO GAME SHOW 2012では『ファンタシースターオンライン2』は、PC版とPS Vita版とスマートフォン版を出展しております。
PC版とPS Vita版は同期プレイが可能で、同じキャラクターを使えるクロスプラットホームになっています。
スマートフォン版では同じキャラクターを使用して、ちょっとした時間にキャラクターを育てたり、ゲームの状況を確認したりという事が出来るようになっています。
プラットフォームの境界を越えるRPGという事で、新しいプレイ、新しいオンラインゲームの世界を体験して頂けると思っています。是非、TGSでプレイしてください。

また、今は別の部署で作っていますが、ファンタシースターオンラインという作品はソニックチームの作品だったんです。現在でもソニックチーム時代のメンバーもしっかり関わっていますし、ゲームの中にもソニックが出てくる事もあるかもしれませんので、ソニックファンの方も要注目してください。

酒井プロデューサーありがとうございました。


★アテンド
さて、セガブースには色々な格好をしたアテンドの方がいるんですよ〜
みなさんの様々なコスチュームを観ているだけでも楽しい〜！要チェック！


★WANTED!!!
TOKYO GAME SHOWの一般日には、【セガブースでソニックを探せ！】企画が行われています！！
ビジネスデーでは「龍が如く5」のバーカウンターにいたソニックが、セガブースのどこかに隠れていますよ！セガブースに来たら是非、探してみてください♪

☆ヒント：SEGA_Appsコーナーのどこかに隠れています。

発見したら、写真を撮ってツイッターでつぶやいちゃいましょう
ハッシュタグも作りました「♯ ソニックを探せ」

答えは、セガイベント用公式ツイッターで発表します！

【セガイベント用公式ツイッターアカウント】

★セガ物販ブース

最後にセガブースから離れ、セガ物販ブースでは以前スタッフコラムにてお知らせした、ソニックの蒔絵シール(ホロ)や、ソニックフィギュアなども販売していますので、ぜひお立ち寄り下さい
それでは、TOKYO GAME SHOW 2012セガブースを楽しんで下さいね

TOKYO GAME SHOW 2011 レポート

2011年09月16日

みなさんこんにちは！
ソニックチャンネル スタッフ★です。



今年もTOKYO GAME SHOW 2011 のセガブース、現地レポートをお届けしますよ☆
17日、18日の参加の際へのご参考に

っと、セガブースの紹介の前に……、今年ももちろんソニックはセガブースに現れるので、皆さんお楽しみに！ソニックを見かけたら、ぜひ20周年のお祝いの言葉をかけてあげてくださいね

それでは、TOKYO GAME SHOW 2011 セガブースのレポートを開始しま〜す

まずは、「龍が如くスタジオ」が開発している完全新作タイトル『バイナリードメイン』！
TPSスタイルのSFアクションシューティングですが、そこは「龍が如くスタジオ」。テーマは、「命」。さて、どんな作品になっているのでしょうか。ぜひ、その目でお確かめください。
会場では、プレイの様子がガラス張りになっており、並んでいる人や通路を歩いている人にプレイヤーの体験がリアルに伝わるようになっています

その横にあるのは、『クロヒョウ2 龍が如く 阿修羅編』。こちらも「龍が如くスタジオ」が手がける新作となります。
前作に引き続き主人公は右京龍也が、東京神室町と新たな舞台大阪で出会う新しいドラマに期待ですよ
ブースは付近は綺麗なお姉さんが沢山居ますが、試遊ブースはまるでクラブのような雰囲気。他には出来ない体験を是非


『セブンスドラゴン2020』は映像出展。
今作の舞台はなんと西暦2020年、TOKYO。あらゆる要素を自分好みにアレンジしてパーティをつくり、すべてのドラゴンを狩り尽くせどんな冒険が待ち受けているのでしょうか（ワクワク）

出展ブースでは、スチューデントスタイルのサムライ女子がお待ちしております。



「セガXプラチナゲームス」最新作…『マックスアナーキー』。
“乱戦格闘アクション”という、これまで前例のない新しいジャンルに挑戦した意欲作で、特殊能力を有するキャラクターたちが繰り広げる乱戦の醍醐味を極限まで詰め込んだ作品となっております。



”今のRPGに、冒険はあるか？---目指すは「無限の冒険」」”…『PHANTSY STAR ONLINE2』。セガオンラインRPGの代表作、「PHANTSY STAR ONLINE」が完全新作として出展中！「ランダムフィールド」や「ハイブリッド カスタム アクション」で新たなオンライン体験を。



『初音ミク　-Project Diva- extend』 アンド　『Project mirai（仮）』
『初音ミク -Project DIVA- 2nd』のシステムを踏襲しつつ、幅広く新たな楽曲とモジュールを収録した最新作>『初音ミク　-Project Diva- extend』。
電子の歌姫”初音ミク”がネンドロイドのデザインで展開される完全新作3Dリズムアクションゲームがニンテンドー3DS™に登場！
TOKYO GAME SHOW 2011では、同じブースで2つの魅力をお楽しみください


ここで、ちょっと一休み
セガ物販コーナーも覗いちゃいましょう☆
セガ物販コーナーでは、「ソニック蒔絵シール」や「20周年企画」の各種サントラ、「ソニック&ぷよぷよ 20周年おめでとうＴシャツ」など、ここでしか手に入らないなグッズを販売しているので、ぜひチェックしてみてください〜
さらに、「ESP」より受注販売される瀬上純モデルのソニックギターも展示中、実物を確認できる数少ないチャンスなので、ぜひこの機会に




さてさて、セガブースに戻りましょう。
次にご紹介するタイトルはこちら。『リズム怪盗R』
セガの完全オリジナルタイトル『リズム怪盗R』は、リズムとストーリーが融合した、新感覚リズムアクションアドベンチャーとして、ニンテンドー3DSで登場します。本作のディレクター中村　俊をTGSで見つけたので、ソニックファンに向けてのメッセージを貰ってきちゃいました
中村
「ソニックチャンネル」をご覧の皆様、こんにちわ。
『リズム怪盗R』ディレクター中村です。
「ソニックチャンネル」を観ている方々はソニック20周年と言う事で、TGSに来られる方も多いと思いますが、その際にはぜひ『リズム怪盗R』のブースにお立ち寄りください。
ソニックのサウンドを担当している大谷智哉が今作の楽曲を制作しおりますし、私も以前ソニックを制作した経験がありますので、ソニックの遺伝子を感じていただけると思います。

さて次に紹介するのは、今年7月にニンテンドーDS版を発売し、このTGS初日にPSP、Wii、ニンテンドー3DS版の発売を発表した『ぷよぷよ！！』ブースのご紹介。
17日・18日は、セガブースではニンテンドー3DS版をお楽しみいただけます。
PSP、Wii版につきましては、ファミリーコーナーでの出展になります。
この『ぷよぷよ！！』もTGSにいた細山田プロデューサーからメッセージを貰ってきました

細山田
「ぷよぷよ」総合プロデューサー細山田です。以前より私もソニックチームの一員として「ぷよぷよ」を制作していますが、今年はぷよぷよ・ソニック共に20周年ということもあるので、ぜひ両方20周年が盛り上がっていければ最高だと思っています。
そして、（すでに知っている方も多いと思いますが）『ぷよぷよ!!』には、「ソニックぷよ」というぷよの種類も出てきますので、ソニックファンの方にも『ぷよぷよ!!』を楽しんで貰えると思っています。

そして、最後のセガブース紹介はお待ちかね、『ソニック ジェネレーションズ』の展示ブース！こちらは、セガブースのほぼ中央にあります
ぜひ皆さんプレイしてくださいね。
そして、こちらも現地にいる飯塚プロデューサー、『白の時空』の宮本ディレクターからもメッセージを貰ってきました

飯塚
『ソニック ジェネレーションズ』プロデューサーの飯塚です。
今年のTGSセガブースでは、『ソニック ジェネレーションズ 白の時空』では17日(土)に「グリーンヒル」、18日(日)に「シティエスケープ」がプレイできます。
『ソニック ジェネレーションズ 青の冒険』では「グリーンヒル」「マッシュルームヒル」が体験できますので、ソニックファンの方々・初めてソニックに触れる方、ぜひ体験してみてください。
また、18日(日)16:20からは、ソニック20周年記念ライブを予定していますので、そちらもぜひお立ち寄りください。


宮本
『ソニック ジェネレーションズ 白の時空』ディレクターの宮本です。今作では、20周年を記念して昔のコアなファンから新しいソニックファンまで、皆が楽しめるように20年分の楽しさを”ギュッ”っと詰め込んだ作品になっていますので、ぜひTGS会場に来ていただき、その楽しさに触れていただければと思います。

いかがでしたか？「ソニックチャンネル」によるTOKYO GAME SHOW 2011 セガブースレポート。
ここで紹介した以外にも、セガパートナーズのタイトルも沢山出展していますので、ぜひ確認してくださいね。
また、セガブースにはソニック以外にも、すけとうだらやラッピーもいるのでチェックしてみてください。


最後にクイズ(？)です
TOKYO GAME SHOW 2011 セガブースでは、数量限定で生産された非売品のソニック20周年フィギアをどこかに展示しているみたい？！
購入はできませんが、そうそう見ることのできない超レアフィギア。ぜひ探してみてくださいね。

ソニック、Wii専用　新作タイトル発表！

2008年10月03日



みなさんこんにちは！　
ソニックチャンネル スタッフ茂呂です。

本日、ソニックのWii専用、新作タイトルが正式にセガより発表となりました！

そのタイトル名は！

『ソニックと暗黒の騎士』

同時に、ティザーサイトもＯＰＥＮいたしましたー！


「ソニックキャラクター イラストコンテスト 2008」にはたくさんのご応募をいただきましてありがとうございました。

採用となった作品は、Wii専用タイトル、『ソニックと暗黒の騎士』内にて使わせていただく予定となっています☆

発売は、２００９年の春を予定しており、現在開発スタッフ一同、鋭意制作中でございます。


新タイトルは、「アーサー王と円卓の騎士」の世界を舞台とした騎士道物語です。
覇業への欲望のため変わり果ててしまった偉大なアーサー王の凶行を止めるため、ソニックが物語の世界に召喚され、見習い騎士として冒険を繰り広げます。
本作では、新感覚の超高速騎士道アクションをお楽しみいただけます。


「東京ゲームショウ2008」では、映像出展も決定いたしました！

セガブースでは、プレイアブル出展を予定している『ソニック ワールド アドベンチャー』と併せ、さまざまなソニックの姿をお楽しみいただけるようになっております。変身するソニック・剣を持ったソニック、それぞれの魅力をゲームショウ会場でお確かめください！

そして、最終日のフィナーレライブには
ジョニーと瀬上純の、”Crush 40”が出演！

今回のＴＧＳは、見どころがたくさんすぎて、私のほうでも興奮気味でございます♪♪♪
