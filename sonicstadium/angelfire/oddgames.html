They called me Sonic...

It's bound to happen isn't it? For every couple of Sonic games released, there's bound to be at least one Sonic game that has been either canned, or been hacked etc. Some games didn't even star Sonic originally, but thanks to home-made technology, that's all change.

Which brings us to this page. Think of this as a twisted alternate 'Games Info Page'; this has information on all the BETA games released so far, all the hacked games and all the info I can find on canned games.
